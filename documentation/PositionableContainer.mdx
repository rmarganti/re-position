# PositionableContainer

`PositionableContainer` is an absolutely positioned container element, a `div`
by default, with rotation and resize handles. It should cover most use cases.

## Example Usage

Below is a basic example of PositionableContainer usage. A more fleshed out
example project is [available here on CodeSandbox](https://codesandbox.io/s/wpkw5jyy7).

## The Transform Box

Note that the transform box is rendered as a sibling to `PositionableContainer`.
This allows it to appear above any other adjacent elements.
